<b>Apachengine</b> - скрипт управления сервером Apache под Linux (веб-панель управления). Есть возможность автоматического добавления, удаления хостов, просмотр php версии, управление phpmyadmin в apache. В предварительной установке будет доступно: PHP7 + Apache2.4 + MySQL5.4 + PHPMyAdmin + Composer. В силу ранней версии, рекомендуется использовать веб-панель управления на виртуальных машинах. <a href="https://github.com/splincode/apachengine">Подробнее</a> <br> <br><br>

<b>Transformicons </b> (перевод сайта + допиливание) - это анимированные значки, выполненные в комбинации с SVG, CSS и HTML. На сайте разработчиков вы можете открыть редактор, благодаря которому вы имеете возможность составить свой набор иконок, в таком случае, у вас полный контроль над имеющимся набором. Анимированные иконки, символы и кнопки написанные на SVG и CSS, на сайте имеется редактор (eng). <b>Kowalski </b>  — онлайн-сервис для ведения списка дел. Сервис будет показывать какие дела я сделал, а какие еще предстоит выполнить. Модель достаточно простая. Ваши дела делятся на основные, то есть, которые вы хотите выполнить в течении данной недели или месяца, и на срочные дела, которые ни в коем случае нельзя отложить на потом. Кнопка «Открыть свой план» работает в современных браузерах поддерживающих HTML5 стандарты. Ваши добавленные дела нигде не будут отображаться, они автоматически сохраняются на стороне браузера за счет работы localStorage. Кнопка «Авторизация. Вконтакте» лишь позволяет вам создавать список дел, который будет закреплен за вашим профилем Вконтакте. <a href="http://splincode.ru/kowalski/">Подробнее</a> <br> <br><br>
