<b>Apachengine</b> - скрипт управления сервером Apache под Linux (веб-панель управления). Есть возможность автоматического добавления, удаления хостов, просмотр php версии, управление phpmyadmin в apache. <b>Kowalski </b> — сервис будет показывать какие дела вы сделали, а какие еще предстоит выполнить. Дела делятся на основные, то есть, которые вы хотите выполнить в течении данной недели или месяца, и на срочные дела, которые ни в коем случае нельзя отложить на потом. Кнопка «Открыть свой план» - дела нигде не будут отображаться, они автоматически сохраняются на стороне браузера. Кнопка «Авторизация Вконтакте» - позволяет вам создавать список дел, которые будут закреплен за вашим профилем. Выпуск: 2014. <a href="http://splincode.ru/kowalski/">Подробнее</a> <br> <br><br>

<b>placeTo</b> — нужно было, чтобы для полей ввода показывались подсказки определенного цвета. В современных браузерах подсказки создаются благодаря атрибуту placeholder. Вот тут я искал готовые решения, но из всех ничего подходящего для себя не нашел. В итоге я написал скрипт за 20 минут, который реализовывал мою задачу, но, чтобы не создавать такой велосипед, созданный мною, я подумал, а не оформить ли в отдельный проект все это, вдруг кому будет интересно. <b>Uply</b> — универсальный загрузчик файлов на сервер HTML5 и CSS3, c возможностью писать собственные расширения в виде плагинов. HTML5 версия: (Firefox 3.6+, IE10+, Chrome 6+, мобильные браузеры), множественная загрузка, дополнительные настройки в виде установки лимитов, типы файлов, поддержка Drag and Drop, настройки стилей, DOM отображения, настройка сервера. Обычный серверный загрузчик: (IE6-9, Opera, Chrome, Firefox, Safari (где нет поддержки HTML5 API FILE и Flash)), одиночная загрузка, лимит на размер файла. Проект не завершен. Выпуск: 2014. <a href="http://splincode.ru/uply/">Подробнее</a> <br> <br><br>

<b>Natali.js (css-препроцессор)</b> - это библиотека, позволяющая решать проблемы с невалидным кодом в различных браузерах, а также данная библиотека является расширением CSS функционала, позволяя писать Javascript код прямо в CSS, ну и еще позволяет вам создавать собственные css-свойства, в целом, всего по немножку. Таким образом, мы с вами можем стать разработчиками CSS, писать минифицированный код, делать все в разы быстрее, используя собственные кроссбраузерные шаблоны. Проект не завершен. Выпуск: 2014. <a href="http://splincode.ru/natali/promo-site/">Подробнее</a> <br> <br><br>
